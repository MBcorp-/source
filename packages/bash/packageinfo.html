Bash is probably the most popular Linux command-line shell. The Busybox shell does a pretty good job of behaving like bash, but the real thing is better.

This package replaces the Busybox implementation in /bin/bash, but it does not replace the one in /bin/sh (it would be necessary to exit the KBOX environment to do this.) To make Bash the default shell without modifying /bin/sh, add it as a command to kbox_shell in the terminal emulator's settings: kbox_shell -c /bin/bash.
